🔥 Firebase Setup (one-time)

To sync products across all devices, you need to connect a free Firebase project. Follow these steps:

  1. Go to console.firebase.google.com → Create a project (free)
  2. Click Firestore Database → Create database → Start in test mode
  3. Go to Project Settings → Your apps → Web app icon → Register app
  4. Copy the firebaseConfig values into the FIREBASE_CONFIG block at the top of index.html
  5. Save and re-open the site — products will now sync across all devices instantly ✓
